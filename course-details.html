Course Description

ok so i originally made these notes for myself and my study group, but figured I'd just turn it into a full thing. if ur professor is just reading off 10-year-old powerpoint slides, this is for u.

we go over the absolute basics first and then dive into javascript. fair warning, JS is where most people get stuck, so I spend extra time there. eventually we build a full react/node app that u can actually show off at career fairs instead of just submitting terminal projects.

disclaimer: i'm still a student so if i take a day to reply on discord, it's bc i have midterms. but my friends also help moderate. there's lots of mini-projects bc honestly watching tutorials does nothing if u don't type the code urself. hope it helps!
